## About The Project

This project is an implementation of a handwritten digit recognizer using a neural network. <br>
A little part of the MNIST database is used to train and test the neural network. <br>
The neural network is implemented in Python using Numpy. <br>

## Prerequisites

You can install prerequisites using the following command:
```sh
pip install -r requirements.txt
```

## Usage

You can run the project using the following commands:
```sh
python main.py
```
